Microsoft пропустила целое поколение с Windows Mobile

Сейчас в Вашингтоне проходит Всемирная конференция партнеров Microsoft, поэтому новостей про Microsoft хватает. Но замечательным стало признание Стива Баллмера, который, выступая перед партнерами, сказал — «We missed a generation with Windows Mobile. We really did miss almost a release cycle».

Я недавно брал в руки телефон с Windows Mobile 6.5 на борту и в очередной раз понял, что не понимаю, как в 2010-м году можно выпускать сенсорные телефоны с таким интерфейсом. Кажется, что Windows Mobile 6 (6.1, 6.5, неважно) оказалась такой же Вистой для мобильных, как и сама Виста для десктопов. Правда, тот факт, что Microsoft сумела выпустить Windows 7, наводит на мысль, что Windows Phone 7 все же окажется таким же прорывом на мобильных.
