Поскольку мне теперь это самому интересно :), встречайте — обновленная версия блога для наладонников. Для пользователей AvantGo — можете пройти напрямую.
Кстати, отличным заменителем AvantGo является тот самый RSS. Просто берете фид, добавляете его в Hand/RSS и читаете — фид обновляется при синхронизации с десктопом. Из недостатков — если сервер фида запнулся при ответе, синхронизация виснет, ридер на палме не показывает дату записи.
