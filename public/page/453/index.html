Яндекс запустил в закрытое тестирование

Яндекс запустил в закрытое тестирование новую версию «Яндекс-денег». Вход пока только по приглашениям. Основное нововведение — наличие веб-версии кошелька, когда деньги хранятся в кошельке на сервере системы и могут быть доступны с любого компьютера через браузер. Красота, между прочим, мой windows-кошелек Яндекс-Денег является для меня предметом постоянной заботы, поскольку это практически единственная, кроме части почты, невосстановимая вещь из необходимых для работы. Если хотите принять участие в тестировании — оставьте свой адрес в комментариях по ссылке выше.
