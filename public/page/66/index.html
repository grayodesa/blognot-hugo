Ищутся разработчики под WordPress

Мопед совершенно не мой, так что сами понимаете. Но одним моим хорошим знакомым требуется WordPress-разработчик. Сразу уточню — не аникейщик, умеющий устанавливать темы и плагины, а разработчик, умеющий сам написать необходимые плагины. Разумеется, надо знать PHP, CSS, Javascript, популярные фреймворки типа jQuery, умение пользоваться Git и разными IDE (например, PHPStorm). Умение использовать распространенные API (Facebook, VK, Twitter и так далее) весьма приветствуется. Умение думать и решать задачи обязательно — не просто накодить, что сказали, а самому придумать решение поставленной задачи.
