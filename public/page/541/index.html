Фантастика, между прочим! В ответ

Фантастика, между прочим! В ответ на мой призывпротестировать Skype откликнулись сразу два человека — спасибо им большое — один из Харькова, другой из Нижнего Новгорода. Слышно действительно замечательно — отличий от обычного городского телефона практически не слышно, причем с отчетливым звуком и т.д. Есть задержки — по типу VoIP, когда голос собеседника доносится с секундной задержкой, но при этом слышимость продолжает быть идеальной. Кстати, по поводу связи с модема — второй из позвонивших звонил с модема, скорость, по его словам, была порядка 44 кБит/сек (вполне стандартно для диалапа), качество — практически безупречное.
