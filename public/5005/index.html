Ответ на это может быть

Ответ на это может быть полезен многим, поэтому вынесу его отдельно.
Решение не мое, придумано оно Сашей Москалюком, я его лишь приложил по месту.
Недавно вышла финальная версия десктопного поиска diskMeta, разработанного нашими ребятами из Харькова, где есть куча полезного — морфология, поддержка разных форматов, нет ограничений по объему и т.д. Строится поисковый индекс и ищется все очень неплохо.
По дефолту она ищет в текстовых файлах, html и doc — я говорю о Lite-версии. Но также в ней имеется возможность добавить любое расширение файла — любой другой файл программа будет индексировать как plain-text.
Теперь видите решение? Элементарно — экспортируем из Бата всю нашу базу в отдельную папку — точнее, экспортируем письма в файлы, — добавляем необходимое расширение в список в программе, запускаем индексацию и наслаждаемся полнотекстовым поиском по всей своей почте. Паллиатив, конечно, но работает.
