Ничто не ново под луной

Ничто не ново под луной :). Вот и вновь всплылатема типографики в постах на блогах. Собственно, я об этом уже писал(давно, правда) и позиция моя остается прежней — заниматься экранным форматированием текста по правилам печатной типографики есть неприкрытое пижонство дизайнеров-перфекционистов, для которых не имеет никакого значение содержание текста. Но, раз уж вам так интересно — вы можете взять user-script для Greasemonkey (расширение для Firefox, позволяющее использовать в браузере пользовательские скрипты) под названием Ctrl+Enter, который умеет, во-первых, отправлять все формы по Ctrl-Enter, при этом форматировать текст по правилам типографики и позволяет использовать вако-коды для форматирования.
