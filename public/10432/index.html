Windows XP, похоже, бессмертна

Microsoft планировала прекратить поддержку возможности даунгрейда на Windows XP сразу после выхода первого сервиспака для Windows 7. По сути, учитывая, что Windows XP уже давно не продаётся отдельно, но установить ее можно, купив Vista или Windows 7, лицензии на которые включают право на даунгрейд на похожие версии XP или Vista, это бы означало прекращение наращивания пользовательской базы Windows XP.

И вот сегодня объявлено о выходе Windows 7 SP1 и одновременно — о продлении поддержки даунгрейда на Windows XP. Покупая компьютер с Windows Vista (это возможно ещё до 22 октября 2010 года) или Windows 7, пользователи имеют полное право установить на нем Windows XP.

Microsoft объясняет это тем, что корпоративным пользователям требуется время на переход на новую ОС и они не могут так быстро перестроить свои системы — что выглядит в принципе логичным, поскольку большинство таких пользователей так и не перешли в массе своей на Vista.
