С утра меня напугала Windows

С утра меня напугала Windows — вместо экрана logon выводила синий экран с сообщением об ошибке. Перегрузился в safe mode и обнаружил источник — вчера поздно ночью ставил заменитель WinGate в виде отечественной разработки lan2net. Удалил — все стало нормально загружаться. Хм…
