Завел тестовый аккаунт на blogspot.com.

Завел тестовый аккаунт на blogspot.com. И, мягко говоря, очень удивился. Во-первых, Blogger API не поддерживает поля title. Т.е. вот так вот просто передать на сервер заголовок из клиента не выйдет.
Во-вторых, я так и не понял, что принимающий сервер делает с кириллицей. Постинг из Windows выдает бред какой-то. Постинг из Линукса прошел успешно — RH 9 построена на Unicode, видимо, поэтому.
Хм…
UPDATE: Воспользовался альфа-версией клиента, поддерживающей Unicode — прошло и нормально отображается.
UPDATE #2: Вот теперь из этой альфа-версии можно и в ЖЖ писать. Только на попытку редактировать сервер возмущается…
