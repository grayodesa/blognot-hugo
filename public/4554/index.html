Возвращаясь (ненадолго) к дискуссии о броузерах — меня больше всего удивило то, что профессиональные дизайнеры (которым, вроде, по должности положено иметь представление о юзабилити и прочем) не раз недоуменно спросили — «А откуда данные по броузерам, с чего вы взяли, что 90% рынка броузеров за IE, у меня у друга Мозилла, а у замдиректора Firebird»…

А все проще гораздо. Вот статистика Рунета, вот — довольно крупного счетчика на Западе. И, наконец, — данные от OneStat.com:

Amsterdam — January 19 2004 — OneStat.com ( www.onestat.com ), the number one provider of real-time web analytics, today reported that Microsoft’s Internet Explorer has a total global usage share of 94.8 percent. Microsoft’s Internet Explorer continues to dominate the global browser market.

Microsoft’s IE 6 is currently the leading browser on the web. Microsoft’s IE 6 global usage has increased with 1.8 percent from 66.3 percent to 68.1 percent since July 2003. Mozilla’s global usage share is 1.8 percent and Opera 7 has a global usage of 0.8 percent.

The global usage share of Apple’s Safari browser has increased with 0.23 percent from 0.25 to 0.48 percent since July 2003.

The most popular browsers on the web are:

  1. Microsoft IE 6.0 68.1%
  2. Microsoft IE 5.5 13.8%
  3. Microsoft IE 5.0 11.8%
  4. Mozilla 1.8%
  5. Opera 7.0 0.8%
  6. Microsoft IE 4.0 0.7%
  7. Safari 0.48%
